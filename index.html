  • Simple yet powerful laser system designer

  • Accelerate design
    • Design calculator:
      The built-in calculator can find solutions for cavities based on user-given conditions.
    • Clear interface:
      A clear and minimalist interface with everything at hand, with the cavity plot as the central piece.
    • Portable:
      No install required, keep it in a usb key and run it everywhere you go.
  • Empower analysis
    • Minimalistic design
      Leaving space for what really matters: the plots.
    • Multiple tools
      Read coordinates, zoom as desired, modify axes...
    • Export
      Plots easily saved as image files (in various formats).
  • Reduce costs
    • Free software:
      Respecting your freedom and costless: no tracking, no ads, 100% science.
    • Multiplatform:
      Use the program or share your designs across multiple operating systems without hassle.
    • Open-source:
      The source code is available to study or modify and adapt to your needs.
